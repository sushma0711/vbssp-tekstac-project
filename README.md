# VBSSP Tekstac Project

This repository contains the projects, programs, and practice exercises completed as part of the Tekstac learning modules. The project includes hands-on implementations covering programming concepts, problem-solving, and application development.

## Features

- Java programming exercises
- Object-Oriented Programming concepts
- SQL and Database-related tasks
- Collections and Exception Handling
- File Handling and JDBC programs
- Spring Framework practice modules
- Mini-project implementations

## Technologies Used

- Java
- SQL
- JDBC
- Spring Framework
- Maven
- MySQL
- Git & GitHub
